## What is Mind Gardens?

Mind Gardens is a system management app built as a replacement for Simply Plural.

Unlike Simply Plural — which was never designed to scale to the feature set it eventually grew into — Mind Gardens is being built from the ground up with that scale in mind. The architecture, the database design, and the feature planning all assume that this will become a complex, feature-rich application. We're not bolting things on after the fact; we're building a foundation that can actually hold weight. This doesn’t invalidate the amazing stuff that Simply plural did for th community

## Planned Features

- Full alter/member management with custom fields, tags, and privacy controls
- Fronting tracking and notifications
- A flexible privacy system so you control exactly who sees what
- Group and subsystem organization
- Board/messaging between alters
- Friend connections between systems
- And much more as development continues

## Tech Stack

- **Frontend:** Next.js + TypeScript + Tailwind CSS
- **Backend:** Supabase (PostgreSQL + Auth + Realtime)
- **Deployment:** Vercel + Cloudflare
